Fully working Whatsapp Clone app using Flutter and Firebase.

The source code is **100% Dart**, and everything resides in the [/lib](https://github.com/shameemreza/reachme/tree/master/lib) folder.

## Features

- Clean and Nice UI/UX
- OTP Sign in system using Firebase
- Passcode auth system
- One to One Chat
- Group Chat (coming soon)
- User Invitation system like whatsapp
- User status like whatsapp
- Audio, Video call with logs and durations
- Share documents like Images, Docs, Videots, Audio record etc
- Shre contact and location
- Giphy image share
- Play video and audio
- End to End encryption
- User profile: Photo, Name and status
- Push notifications
- User Block system
- Delete Chat system
- Support both ANdroid and IOS
- And probably many more.

## Requirements

- Flutter
- Firebase
- Giphy API
- Agora SDK (Audio and Video Call)
- Admob Ads
